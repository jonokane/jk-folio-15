Description
The marketing face to Fluence's powerful platform, the Fluece website posed an interesting challenge, as its primary function is to inform and captivate those that may be interested in using the platform.
link http://fluence.io
  • The Fluence welcome page. The very first thing someone sees when they visit Fluence. We ran a lot of tests to determine that the sign up fields are most useful when placed right on this very page. This page also features videography and photography shot by me.
  • The Fluence Lists feature landing page. Showing various lists. Using Masonry.js for dynamic layout.
  • The Fluence Profile page. Every Fluence user gets a profile page, which they can customize.
